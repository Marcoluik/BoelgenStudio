# Bølgen Studio

En moderne, SEO-venlig hjemmeside bygget med Astro.

## Kom i gang

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Kontakt & nyhedsbrev

Sæt `PUBLIC_CONTACT_EMAIL` til den indbakke, der skal modtage beskeder (fx `kontakt@ditdomæne.dk`). Formularen bruger [FormSubmit](https://formsubmit.co) — ingen API-nøgle; du bekræfter e-mailen første gang du modtager en besked.

Hvis variablen er tom, bruges **Netlify Forms** (kun relevant hvis siden deployes på Netlify). Se `.env.example`.
